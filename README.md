# t_app_pro

原创税务服务风格 Flutter App V1。

## 页面
- 登录 / 验证码体验页
- 首页仪表盘
- 收入纳税明细
- 收入详情
- 搜索和筛选
- 图表统计
- 税额计算器
- 申诉演示
- 我的 / 设置

## GitHub 自动打包 APK
推送到 GitHub 后进入：

Actions -> Build Android APK -> 最新任务 -> Artifacts -> 下载 `t-app-pro-release-apk`

里面就是 `app-release.apk`。

## 本地运行
```bash
flutter pub get
flutter run
```
